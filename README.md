# Initial Setup
- ```npm i``` to install missing depdencies.
- ```.env``` file with:
  - `DISCORD_TOKEN=YOUR_BOT_TOKEN`
  - `CLIENT_ID=YOUR_BOT_ID`
  - `GUILD_ID=YOUR_SERVER_ID`
- **The bot will not authenticate without those 3 lines.**
